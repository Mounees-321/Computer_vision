# Quick Start Guide

## Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Mounees-321/Computer_vision.git
cd Computer_vision
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

3. **Start Jupyter:**
```bash
jupyter notebook
```

## Try These Notebooks First

### 1. **Start Here: Basic Image Processing**
- Open: `basic_image_processing.ipynb`
- Learn: Image loading, display, and basic operations

### 2. **Feature Detection Demo**
- Open: `HArris (2).ipynb`
- See: Corner detection, edge detection, line detection all in one

### 3. **Object Detection**
- Open: `obj_detection.ipynb`
- Try: Finding objects in images using SIFT features

### 4. **Motion Tracking**
- Open: `Lucas_kanade_(optical flow).ipynb`
- Watch: Track moving objects in video

### 5. **Image Classification**
- Open: `Image_classify.ipynb`
- Learn: Machine learning for image recognition

## Need Help?

## Common Issues

**"Module not found"**: Install requirements with `pip install -r requirements.txt`

**"Image not found"**: Replace image paths in notebooks with your own images

**"Video not found"**: Add your own video files or update the paths in video processing notebooks
